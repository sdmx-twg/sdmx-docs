# SDMX 3.1 Documentation Discovery Map

Curated, version-scoped entry points for SDMX 3.1 documentation and canonical
source repositories.

## Getting Started

- [SDMX 3.1 landing page](https://sdmx-twg.github.io/sdmx-docs/3.1/): Primary
	entry point for all SDMX 3.1 technical documentation.

## Framework

- [SDMX Framework](https://sdmx-twg.github.io/sdmx-docs/3.1/framework/):
	Foundation concepts, architecture, and shared specification context.

## Information Model

- [Information Model](https://sdmx-twg.github.io/sdmx-docs/3.1/information_model/):
	Core SDMX information structures, artefacts, and relationships.

## Registry and Logical Interfaces

- [Registry](https://sdmx-twg.github.io/sdmx-docs/3.1/registry_specification/):
	Registry repository model, architecture, and service behavior.
- [Logical Interfaces](https://sdmx-twg.github.io/sdmx-docs/3.1/logical_interfaces/):
	Interface semantics and interaction patterns across SDMX services.

## Technical Notes

- [Technical Notes](https://sdmx-twg.github.io/sdmx-docs/3.1/technical_notes/):
	Clarifications, implementation guidance, and supporting technical details.

## REST API

- [REST API](https://sdmx-twg.github.io/sdmx-docs/3.1/rest_api/): SDMX REST
	interface guidance for querying structures, data, and metadata.

## Transmission Formats

- [SDMX-ML](https://sdmx-twg.github.io/sdmx-docs/3.1/ml/): XML-based
	transmission format reference and message documentation.
- [SDMX-JSON](https://sdmx-twg.github.io/sdmx-docs/3.1/json/): JSON-based
	transmission format reference for SDMX exchange workflows.
- [SDMX-CSV](https://sdmx-twg.github.io/sdmx-docs/3.1/csv/): CSV-based
	transmission format reference for tabular exchange scenarios.

## Official SDMX

- [SDMX home](https://sdmx.org/): Official SDMX website for standards and
	governance context.

## Canonical Repositories

- [sdmx-docs](https://github.com/sdmx-twg/sdmx-docs): Version context: SDMX 3.1
	published documentation set and cross-component integration.
- [sdmx-im](https://github.com/sdmx-twg/sdmx-im): Version context: SDMX 3.1
	information model specification.
- [sdmx-registry](https://github.com/sdmx-twg/sdmx-registry): Version context:
	SDMX 3.1 registry specification and repository architecture.
- [sdmx-rest](https://github.com/sdmx-twg/sdmx-rest): Version context: REST API
	v2.2 used in the SDMX 3.1 documentation suite.
- [sdmx-ml](https://github.com/sdmx-twg/sdmx-ml): Version context: SDMX-ML
	v3.1 transmission format specification.
- [sdmx-json](https://github.com/sdmx-twg/sdmx-json): Version context:
	SDMX-JSON v2.1 transmission format specification.
- [sdmx-csv](https://github.com/sdmx-twg/sdmx-csv): Version context: SDMX-CSV
	v2.1 transmission format specification.
